# networkoptimization_GAvsPSO

Để chạy chương trình chỉ cần chạy file Main.py 

Các tính năng và thao tác có thể làm khi mở giao diện :
1. Chọn thuật toán Genetic algorithm hoặc Bat algorithm.

2. Sau khi đã chọn thuật toán có thể tuỳ chỉnh thông số tuỳ thích, hoặc giữ mặc định

3. Chọn đỉnh đầu và đỉnh cuối

4. Chọn mạng, sẽ mở folder ra cho bạn chọn file csv mạng theo dạng
v_from,v_to,weight
có thể cho file vào folder Graph 

5. Chạy tối ưu

6. Hiển thị sơ đồ dựa theo file csv

7. Compare result : để so sánh kết quả của thuật toán tối ưu bầy đàn hạt với thuật toán edmond_karp
